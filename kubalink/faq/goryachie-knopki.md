# Горячие кнопки

* Использование "**Ctrl + Левый Click**" в схеме коммутации сооружения связи на волокне или порту оборудования позволяет мгновенно переключить этот волоконный/портовый соединитель, избегая необходимости нажимать на дополнительный элемент коммутации для переключения.
* Комбинация "**Ctrl + Z**" в схеме коммутации сооружения связи отменяет последнее действие.
* При нажатии "**Ctrl + Левый Click**" на названии типа задания в карточке задания открывается форма настройки соответствующего типа задания.
* С помощью "**Alt + Левый Click**" на карте покрытия, если кликнуть на значок сооружения связи, открывается схема коммутации данного сооружения.
* "**Alt + Левый Click**" на полигоне здания на карте покрытия открывает план этого здания.
