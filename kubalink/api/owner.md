# owner

```
Дополнительные параметры:
action - подкатегория запроса
```

## Подкатегории

### **add**

```
Описание: Добавление 
Обязательные параметры:
 name - наименование
Дополнительные параметры:
 comment - заметки
 cost - размер оплаты собственнику
```

### **bind\_building**

```
Описание: Указание собственника для здания
Обязательные параметры:
 owner_id - id собственника
 building_id - id здания
```

### **bind\_fiber**

```
Описание: Указание собственника для ВОЛС
Обязательные параметры:
 owner_id - id собственника
 fiber_id - id ВОЛС
```

### **bind\_node**

```
Описание: Указание собственника для объекта инфраструктуры
Обязательные параметры:
 owner_id - id собственника
 node_id - id объекта инфраструктуры
```

### **delete**

```
Описание: Удаление объекта
Обязательные параметры:
 id - id объекта
```

### **edit**

```
Описание: Редактирование объекта
Обязательные параметры:
 id - id объекта
Дополнительные параметры:
 name - наименование
 comment - заметки
 cost - размер оплаты собственнику
```

### **get**

```
Описание: Список объектов
Необязательные параметры:
 id - перечень id объектов (через запятую)
```

### **unbind\_building**

```
Описание: Исключение собственника у здания
Обязательные параметры:
 owner_id - id собственника
 building_id - id здания
```

### **unbind\_fiber**

```
Описание: Исключение собственника у ВОЛС
Обязательные параметры:
 fiber_id - id ВОЛС
```

### **unbind\_node**

```
Описание: Исключение собственника у объекта инфраструктуры
Обязательные параметры:
 node_id - id объекта инфраструктуры
```
