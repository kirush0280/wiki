# newin

{% code overflow="wrap" %}
```
Дополнительные параметры:
subcat - подкатегория запроса (возможное значение: new (пустое значение этого параметра равняется new))
citycode - ID населенного пункта, по которому подаётся заявка. Будет открыт выбор улиц. Если пусто - будет открыт выбор населенного пункта.
cityignore - ID населенных пунктов, которые требуется игнорировать и не предлагать в выборе. Несколько населенных пунктов - через запятую.
map - флаг (1/0) - открывать ли окно выбора карты. Населенный пункт должен быть указан на карте - тогда будет понятно какую карту открывать и где её центровать.
```
{% endcode %}

Пример запроса:

```
api.php?key=apikey&cat=newin&map=1
```
