## Подкатегории


### **send\_email**

```
Описание: Отправка e-mail
Обязательные параметры:
 recipient - адрес получателя
 subject - тема
 body - текст
Необязательные параметры:
 is_html - флаг - тело сообщения в html-формате
```
